# Javascript circular slider component

### Version

1.1.0

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve

To serve in the browser - Runs webpack-dev-server

```sh
$ npm start
```

### Build

Compile and build

```sh
$ npm run build
```
